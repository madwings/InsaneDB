###################
What is InsaneDB
###################

InsaneDB is an PHP Database Library forked from the Database Library of CodeIgniter 3. 
Its goal is to remove limitation existing in the upstream version and add aditional
functionlaities while keeping sync with CodeIgniter. It can be used standalone or
as drop in replacement for the original CodeIgniter library.
